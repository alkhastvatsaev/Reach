# 🎯 Reach : Stratégie "Sniper" (Qualité > Quantité)

> **Note de l'architecte :** Ce document définit comment transformer Reach d'un automate d'envoi de mails en une arme redoutable pour décrocher une alternance en BTS SIO / Dev.

---

## 🏗️ Phase 1 : Infrastructure & Réputation (Le "Warmup")

L'erreur n°1 est d'utiliser son adresse Gmail personnelle. Pour réussir, il faut une infrastructure dédiée.

- [ ] **Nom de domaine secondaire** : Acheter un domaine comme `alkhast-dev.fr` ou `vatsaev-tech.com`.
- [ ] **Email dédié** : Créer une adresse pro (ex: `contact@alkhast-dev.fr`).
- [ ] **Warmup automatique** : Utiliser un outil (Instantly.ai ou Lemlist) pendant 14 jours pour "chauffer" l'adresse et éviter les filtres Office 365.

## 🧠 Phase 2 : Le Contenu "Anti-Robot"

Les recruteurs Tech détectent l'IA à 10km. L'IA doit être ton assistant de recherche, pas ton rédacteur.

- **Le Hook "Hard"** : Au lieu de "Je suis passionné", utiliser "J'ai vu que vous utilisez [Techno X] sur votre site, j'ai justement conçu [Projet Y] qui règle [Problème Z]".
- **L'Appel à l'action (CTA)** : Proposer un call de 10 min, pas un entretien de 2h.
- **La Signature Dynamique** : Inclure un lien vers le dépôt GitHub de **Reach** lui-même.

## 🎥 Phase 3 : Vendre "Reach" aux recruteurs

Reach est la preuve de tes compétences en BTS SIO (SLAM). Voici comment le présenter en entretien :

1. **Architecture** : "J'ai orchestré une stack Next.js avec 5 agents IA spécialisés qui s'occupent de l'analyse, du scoring et de la rédaction personnalisée."
2. **Data-Scraping** : "J'ai développé un crawler hybride qui scanne les mentions légales et les API (Apollo/Clearbit) pour identifier les décideurs techniques."
3. **Product Mindset** : "Plutôt qu'envoyer 100 mails génériques, j'ai codé un outil qui me permet de n'envoyer que 5 candidatures ultra-qualitatives par jour."

## 🚀 Le Workflow "Sniper" dans l'App

1. **Discovery** : Charger uniquement 5 cibles par jour.
2. **Deep Scan** : Identifier le CTO ou le Lead Dev.
3. **Review Manuelle** : Utiliser la fenêtre preview pour modifier 1 ou 2 phrases et ajouter une touche **100% humaine** (ex: mentionner un post LinkedIn du recruteur).
4. **Follow-up** : Si pas de réponse après 4 jours, envoyer un message LinkedIn court.

## 🛠️ To-Do technique immédiate :

- [ ] **Ajouter un Pixel de Tracking** : Savoir si le mail est ouvert.
- [ ] **Lien Loom** : Enregistrer une démonstration de 30s de l'outil et l'inclure dans le mail.

**Rappel : Ta valeur n'est pas dans le mail, elle est dans le fait que tu as été capable de CODER l'outil qui envoie le mail.**
